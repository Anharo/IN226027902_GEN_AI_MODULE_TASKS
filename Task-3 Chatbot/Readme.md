# 🤖 AI Chatbot using Hugging Face Transformers

## 📌 Overview

This project is a console-based AI chatbot built using pre-trained transformer models from Hugging Face.
The chatbot can understand user input and generate meaningful, human-like responses in real time.

It demonstrates the practical use of **Natural Language Processing (NLP)** and **transformer-based language models**.

---

## 🎯 Objective

* Build a conversational AI chatbot
* Use pre-trained transformer models
* Implement prompt-based response generation
* Maintain continuous interaction with users

---

## 🧠 Technologies Used

* Python 🐍
* Hugging Face Transformers 🤗
* PyTorch 🔥
* Jupyter Notebook / VS Code

---

## ⚙️ Model Used

**FLAN-T5 (google/flan-t5-base)**

Why FLAN-T5?

* Instruction-tuned model
* Generates accurate and meaningful responses
* Faster and more reliable than traditional conversational models like DialoGPT

---

## 🔄 Chatbot Workflow

User Input → Tokenization → Model Processing → Response Generation → Output → Loop

---

## 🚀 Features

* Interactive console-based chatbot
* Uses transformer-based NLP model
* Generates dynamic responses
* Maintains conversation flow
* Exit command support (`exit` / `quit`)

## ▶️ Usage

Run the Python script or Jupyter Notebook:

```bash
python chatbot.py
```

Example interaction:

```
Chatbot: Hello! I am your AI assistant. How can I help you today?

You: What is Artificial Intelligence?
Chatbot: Artificial Intelligence is the simulation of human intelligence by machines.

You: Who created Python?
Chatbot: Python was created by Guido van Rossum.

You: exit
Chatbot: Goodbye!
```
